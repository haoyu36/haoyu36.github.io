文件读写

一：文件

文件是储存在外部介质上的数据或信息集合。如程序中的源程序，数据中保存的数据，图像中的像素数据

二进制文件是信息按照非字符但特定格式形成的文件，由于没有统一的字符编码，只能当做字节流，而不能看作字符串。照片，音乐，视频，计算机程序都采用二进制存储，更加节省空间且表示更为精确

通常文件会包含以下信息：名字，类型（类型表明了文件中包含了哪种文件的数据），属性，位置，大小等等

Python 中，文件操作按以下顺序进行

  1. 打开文件建立文件对象
  2. 通过文件对象对文件进行操作
  3. 关闭文件

二：open

打开文件

使用内置函数 open() 打开文件，如果文件无法打开，引发 OSError

mode 是一个可选的字符串，用于指定打开文件的模式。默认为 ‘r’，表示在文本模式下打开。在文本模式下，如果未指定编码，则使用的编码取决于平台

mode

mode 说明
r open for reading (default)
w open for writing, truncating the file first
x create a new file and open it for writing
a open for writing, appending to the end of the file if it exists
b binary mode
t text mode (default)
+ open a disk file for updating (reading and writing)

模式可以组合，比如 rb 就表示读取二进制文件

读写方法

方法 说明
f.read(size=1) 读取整个文件内容，如果给出参数，读入前 size 长度的字符串或字节流
f.readline(size=1) 读取一行内容，加参数
f.readlines(hint=1) 读取所有行，或读取 hint 行
f.write(s) 向一个文件写入字符串或字节流
f.writelines(lines) 将一个元素全为字符串的列表写入文件
f.seek(offset) 改变当前文件操作指针位置
f.tell() 返回文件当前指针

三：使用

编码与换行符

文件的读取和写入采用的都是系统默认的文本编码方式，可以通过 sys.getdefaultencoding() 查询，可以对文本文件的编码进行设置

关于换行符的识别，在 UNIX 和 Windows 上是不同的。默认情况下，Python 能识别所有常见的换行格式

在读取文件时会将换行符转换为一个单独的 \n 字符，在输出换行符时会被转换为默认的换行符，x.strip() 函数去除空行

with open('somefile', 'rb') as f:
    data = f.read()

# 使用 wb 写入二进制数据
with open('somefile', 'wb') as f:
    f.write(b'hello world')

读取二进制数据时所有数据应以字节串（byte string）的形式返回而不是文本字符串

同样的当写入二进制数据时，数据必须是以对象的形式来提供，而且该对象可以将数据已字节形式暴露出来（字节串，bytearray 等）

在进行索引和迭代操作时，字节串会返回代表该字节的整数值而不是字符串

def read_in_chunks(file_obj, chunk_size = 2048):
    '''默认块大小：2KB'''
    while True:
        data = file_obj.read(chunk_size)  # 每次读取指定的长度
        if not data:
            break
        yield data

with open('filename', 'r', encoding = 'utf-8') as f:
    for chuck in read_in_chunks(f):
        do_something(chunk)


# 读取遍历文件的每一行
with open('xxx', 'r', encoding='utf-8') as f:
    for i in f:
        do_something(i)
